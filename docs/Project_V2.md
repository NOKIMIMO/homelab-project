# HomeStock 

HomeStock est un server maisons capable de Host efficacement les différents fichiers des utilisateurs (Cloud auto Hoste). L'application sera accompagné d'interface d'affichage spécialisé aux différents type de fichier et media.

## Son but

- Stockage de photo en cloud partagé
- Affichage spécialisé des différents formats 
- Gestion des accés de ressources par groupe utilisateur
